<h1>🌐 Enterprise Network Simulation – Cisco Packet Tracer</h1>

<h2>📌 Project Description</h2>
<p>
This project is a structured enterprise network simulation built using Cisco Packet Tracer. 
It demonstrates the practical implementation of core networking concepts such as IP addressing, 
subnetting, VLAN configuration, routing, and DHCP services within a simulated business environment.
</p>

<p>
The objective of this project is to design and configure a scalable, logically segmented, 
and fully functional network infrastructure that enables secure and efficient communication 
between multiple departments.
</p>

<p>
This simulation reflects real-world networking scenarios and strengthens hands-on configuration 
skills using Cisco CLI.
</p>

<hr>

<h2>🛠 Tools & Technologies</h2>
<ul>
  <li>Cisco Packet Tracer (.pka)</li>
  <li>Router CLI Configuration</li>
  <li>Switch Configuration</li>
  <li>VLAN Implementation</li>
  <li>Subnetting & IP Addressing</li>
  <li>DHCP Configuration</li>
  <li>Static / Inter-VLAN Routing</li>
  <li>Network Testing & Troubleshooting</li>
</ul>

<hr>

<h2>⚙️ Network Features Implemented</h2>

<h3>🔹 IP Addressing & Subnetting</h3>
<ul>
  <li>Designed and assigned structured IP address scheme</li>
  <li>Configured interfaces with appropriate IP addresses</li>
  <li>Verified network connectivity</li>
</ul>

<h3>🔹 VLAN Segmentation</h3>
<ul>
  <li>Created VLANs to separate departments</li>
  <li>Assigned switch ports to specific VLANs</li>
  <li>Configured trunk links where required</li>
</ul>

<h3>🔹 Routing Configuration</h3>
<ul>
  <li>Enabled inter-VLAN communication</li>
  <li>Configured router interfaces/subinterfaces</li>
  <li>Ensured cross-network communication</li>
</ul>

<h3>🔹 DHCP Configuration</h3>
<ul>
  <li>Configured automatic IP address allocation</li>
  <li>Reduced manual configuration errors</li>
  <li>Verified proper client IP assignment</li>
</ul>

<h3>🔹 Network Testing</h3>
<ul>
  <li>Used ping commands to test connectivity</li>
  <li>Identified and resolved configuration issues</li>
  <li>Validated end-to-end communication</li>
</ul>
